# Terra Agraria: From Soil to Stars (v2.1.0)

A long-term Minecraft 1.21.1 NeoForge modpack built around TerraFirmaCraft.

## Quick Start
1. Install Prism Launcher (https://prismlauncher.org)
2. Add Instance → Import from ZIP → select this zip
3. Allocate 4GB+ RAM
4. Launch!

## What's New in v3.0.1
- Better Questings
- Added Apotheosis + Apothic Attributes, Enchanting, Spawners
- Added When Dungeons Arise (epic dungeon structures)
- Add-ons
